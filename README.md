## Controls

- **Click** to lock pointer and start navigation
- **WASD** or **Arrow Keys** to move
- **Mouse** to look around
- **ESC** to unlock pointer

## Art Pieces

1. **Abstract Geometry** - Colorful geometric shapes
2. **Concentric Dreams** - Layered circular forms
3. **Golden Spiral** - Dynamic spiral pattern
4. **Floating Cosmos** - Suspended spheres
5. **Triple Helix** - Intertwined helical structure
6. **Ocean Waves** - Flowing wave patterns

## Tech Stack

- Three.js (r160)
- PointerLockControls
- WebGL rendering
- ES6 modules

## Development

Simply open `index.html` in a modern web browser. No build process required!
